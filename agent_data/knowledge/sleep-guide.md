# 睡眠改善指南：睡不着时可以这样做

## 一、睡眠卫生基本原则
1. **固定起床时间**：无论前一晚睡了多久，每天都在同一时间起床
2. **床只用来睡觉**：不要在床上工作、刷手机、吃饭
3. **睡前1小时远离屏幕**：手机、电脑的蓝光会抑制褪黑素分泌
4. **下午3点后不午睡**：避免消耗晚上的睡眠驱动力
5. **睡前避免咖啡因和酒精**：两者都会干扰睡眠质量

## 二、20分钟法则（睡不着别硬躺）
如果你躺在床上超过20分钟仍然清醒，或开始感到烦躁：
1. **果断起床**，离开卧室
2. 去另一个安静、灯光昏暗的房间
3. 做一些**温和、单调、不费脑**的事：
   - 读一本平淡的纸质书
   - 听舒缓的轻音乐
   - 简单拉伸
4. **绝对不要**看手机或电脑
5. 等再次感到困意时，再回到床上
6. 如果回去后又清醒超过20分钟，重复以上步骤

**原理**：打破"床=清醒+焦虑"的负面条件反射，重新建立"床=快速入睡"的联结。

## 三、刺激控制法（重建床与睡眠的联系）
1. 只有感到困倦时才上床
2. 床只用于睡眠（和亲密关系），不进行其他活动
3. 躺下后超过15-20分钟睡不着就起床离开
4. 每天固定时间起床
5. 白天不补觉、不打盹

## 四、睡前放松小仪式
睡前30分钟可以做：
- 温水泡脚10-15分钟
- 渐进式肌肉放松（从脚到头的"紧张-放松"循环）
- 腹式呼吸5分钟
- 听一段白噪音或自然声音（雨声、海浪声）

## 五、关于失眠的心态调整
- 偶尔睡不好是正常的，不会毁掉第二天
- 越强迫自己"必须睡着"，越容易清醒
- 把目标从"睡着"改成"让身体休息"，躺着放松本身就是一种恢复
- 如果失眠持续数周且严重影响白天状态，建议咨询专业医生
