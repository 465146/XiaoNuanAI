# 焦虑应对策略：当紧张来临时可以做的事

## 一、腹式呼吸法（最快镇静）
当你感到心跳加快、呼吸急促时，试试这个：
1. 找个舒服的姿势坐下或躺下，一只手放在腹部
2. 用鼻子深深吸气，感受腹部像气球一样鼓起来（约4秒）
3. 屏住呼吸片刻
4. 用嘴巴缓慢呼气，感受腹部回落（约6秒）
5. 重复5-10次，把注意力完全放在呼吸的气流上

**小提示**：呼气比吸气更长，能激活副交感神经，帮助身体进入放松状态。

## 二、5-4-3-2-1 接地技术（把注意力拉回当下）
当思绪乱飞、控制不住担忧时，用感官把自己"锚定"在现实中：
- **看**：找出周围5样东西，在心里说出它们的名字（如：台灯、水杯、窗帘……）
- **摸**：触摸4种不同的物体，感受其质地（如：衣服的柔软、桌面的光滑）
- **听**：仔细听，识别3种不同的声音（如：空调声、键盘声、远处车声）
- **闻**：识别2种你能闻到的气味（如：咖啡香、空气清新剂的味道）
- **尝**：吃一小块糖果或喝一口水，专注于它的味道

## 三、渐进式肌肉放松（释放身体紧张）
焦虑时身体会不自觉地紧绷，这个练习帮你主动"松开"：
1. 从脚开始：用力蜷紧脚趾，保持5-10秒 → 突然放松，感受松弛感
2. 依次向上：小腿 → 大腿 → 腹部 → 手部（握拳）→ 手臂 → 肩膀（耸肩）→ 面部（皱眉、挤眼）
3. 每个部位：紧张5-10秒 → 彻底放松15-20秒
4. 全程保持平稳呼吸，不要在紧张时憋气

**什么时候用**：睡前、感到肩颈僵硬时、或任何注意到身体紧绷的时刻。

## 四、安静场景想象（给大脑放个短假）
闭上眼睛，用1-2分钟想象一个让你感到安全、平静的地方。尽可能调动所有感官：
- 看到什么颜色和光线？
- 听到什么声音？
- 闻到什么气味？
- 皮肤感受到什么温度？

不需要让想象"完美"，只要让自己在那个画面里待一会儿就好。
